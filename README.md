# CODSOFT Internship - Data Science Projects

## Overview
This repository contains 3 completed tasks for the CODSOFT Data Science Internship:
1. Iris Flower Classification
2. Titanic Survival Prediction
3. Sales Prediction

---

## 1. Iris Flower Classification
**Objective:** Classify Iris flowers into species (Setosa, Versicolor, Virginica)  
**Dataset:** Iris dataset (150 samples, 4 features)  
**Model:** Random Forest Classifier  
**Accuracy:** ~97%  
**Notebook:** `Iris_Classification/iris.ipynb`  

---

## 2. Titanic Survival Prediction
**Objective:** Predict whether a Titanic passenger survived  
**Dataset:** Kaggle Titanic Dataset (`train.csv`)  
**Model:** Random Forest Classifier  
**Accuracy:** ~80%  
**Notebook:** `Titanic_Prediction/titanic.ipynb`  

---
